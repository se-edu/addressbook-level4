* This is a desktop TuitionConnect application. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).

== Features

* Tutor is able to add, delete, edit any tuition or personal task into the schedule.
* Has a calendar that makes viewing of task easier.
